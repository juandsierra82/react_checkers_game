## Table of Contents

Dependencies `Node.js 5 or higher` and `Npm`

Install `npm install`

Run `npm start`

Then point your browser to localhost:3000

Works by dropping text files into a droppable area with a list of moves.

Text file format 1,1,1,1\n2,2,2,2
